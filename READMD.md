# elsa
:zap: elsa（eleme simple admin）基于 element-ui 封装 el-form，el-table 等组件，适用于快速开发后台管理项目。

## quickstart

## Feature

- [ ] easy-el-form
  - [x] 自定义布局，遵循 el-row/el-col 配置
  - [x] 表单数据回显
  - [x] 额外信息提示
  - [x] 表单提交时校验
  - [x] nest 嵌套对象的解析
  - [x] 表单项关联功能
    - [ ] 查询、修改、添加展示不同特性表单
    - [x] 禁用/可用
    - [x] 显示/隐藏
    - [x] 动态修改校验规则
    - [x] 可清空表单项值
    - [ ] 内部支持 $store 
  - [ ] 表单项
    - [x] input
      - [ ] 历史信息下拉展示
    - [x] select
      - [x] 支持延迟加载
    - [x] checkbox
    - [x] radio
    - [x] 级联选择
    - [x] date-picker
      - [x] 不同表单项之间的时间跨度限制
    - [x] upload
